# ЕГЭ-навигатор

Современный образовательный веб-сайт для подготовки к Единому Государственному Экзамену.

## Функциональность

1. **Главный экран** с заголовком и описанием сайта
2. **Навигация** по разделам сайта
3. **Раздел "Материалы"** с фильтрацией по предметам:
   - Математика
   - Русский язык
   - Физика
   - Информатика
   - Обществознание
4. **Раздел "Калькулятор баллов"**:
   - Ввод баллов по 5 предметам
   - Автоматический расчет общей суммы
   - Оценка результатов

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, CSS variables)
- JavaScript (Vanilla)
- Font Awesome для иконок
- Шрифт Inter от Google Fonts

## Запуск проекта

Просто откройте файл `index.html` в браузере.

## Структура проекта

- `index.html` - главный файл HTML
- `styles.css` - стили сайта
- `script.js` - логика сайта (навигация, фильтрация, калькулятор)
- `README.md` - документация

## Особенности реализации

1. Адаптивный дизайн (работает на мобильных устройствах)
2. Плавные анимации и переходы
3. Интерактивные элементы с обратной связью
4. Современный чистый дизайн
5. Простая навигация
